# Changelog

### Added
- 🚀 初始版本发布。
- ✨ 支持多用户订阅源管理。
- ✨ 支持 Token 访问控制与轮换。
- ✨ 实现内容正则过滤（Include/Exclude）。
- ✨ 实现历史版本快照与一键回滚。
- ✨ 集成 Cloudflare Turnstile 验证码。
- ✨ 添加 Webhook 通知支持。
- 🎨 全新移动端优先的 UI 设计，支持深色模式。

### Security
- 🔒 实现 SSRF 防护机制。
- 🔒 增加登录暴力破解防护。
- 🔒 全局 API 速率限制。
